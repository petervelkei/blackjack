\doxysubsection{Detailed Description}
A játékos egyenlegét és fogadásait kezelő osztály.

Ez az osztály kezeli a játékos teljes egyenlegét és a jelenlegi fogadását. Lehetővé teszi a fogadások hozzáadását, a nyeremények és veszteségek kezelését, valamint a fogadások visszaállítását.
