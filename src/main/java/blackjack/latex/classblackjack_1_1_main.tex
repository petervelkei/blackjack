\doxysubsection{Detailed Description}
A Main osztály, amely a Blackjack játék indításáért felelős.

Ez az osztály tartalmazza a main metódust, amely a program belépési pontja. A main metódus létrehoz egy új Game objektumot, és elindítja a játékot.
